## Installation

1. Clone the repository:

```bash
git clone https://github.com/annalin2000/react-portfolio.git
cd react-portfolio
```

2. Install dependencies:

```bash
npm install
```

## Usage

1. Start the server:

```bash
npm run dev
```

2. Open your browser and navigate to:

```
http://localhost:5173
```
